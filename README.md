# 🏓 Philly Pong

A full-stack web application designed to help players discover, filter, and explore table tennis locations across the Greater Philadelphia area.

---

## 🚀 Features

* **Dynamic Data Modeling:** Structured TypeScript interfaces (`Table.ts`) ensuring strict type safety from backend models to UI components.
* **Instant Filtering:** Real-time client-side filtering system allowing users to slice data by environment (Indoor/Outdoor) and cost (Free/Paid).
* **Master-Detail View:** Clean card-based layout featuring expandable details (pricing info, exact address, hours, net quality, and amenities like paddle/ball availability).
* **Cross-Origin API Integration:** Express backend serving RESTful JSON endpoints consumed asynchronously by a modern React/Vite front-end.

---

## 🛠️ Tech Stack

* **Front-End:** React, TypeScript, Vite, CSS / Inline Styling
* **Back-End:** Node.js, Express, REST API
* **Architecture:** Full-stack decoupled client-server model
